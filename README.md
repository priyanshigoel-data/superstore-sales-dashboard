# Executive Sales Performance Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard built using the Sample Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, regional performance, and product-level analysis.

---

## 📂 Dataset

- Sample Superstore Dataset
- Time Period: 2014–2017

---

## 🛠️ Tools Used

- Power BI
- Power Query

---

## 📊 Dashboard Features

- Total Sales
- Total Profit
- Total Orders
- Average Delivery Days
- Yearly Sales Trend
- Revenue by Category & Sub-Category
- Profit by Category
- Customer Segment Distribution
- Profit by State & Region
- Top Performing Products
- Interactive Year Filter

---

## 💡 Key Insights

- Technology generated the highest profit.
- Consumer segment contributed nearly 50% of total sales.
- Western region recorded the strongest sales performance.
- Furniture generated comparatively lower profits.
- Sales showed an upward trend over the years.
